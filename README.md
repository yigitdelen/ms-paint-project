# MS Paint Clone 🎨

A robust, object-oriented MS Paint clone built with C# and WPF. This application provides a modern and intuitive interface for fundamental vector-based drawing operations.

## Overview

This project demonstrates the practical application of software architecture and design patterns in desktop application development. It recreates the classic MS Paint experience while leveraging the modern capabilities of the Windows Presentation Foundation (WPF) framework.

## Features

*   **Versatile Drawing Tools:** Includes essential tools such as Pen, Eraser, and various geometric shapes.
*   **Dynamic Tool Selection:** Seamlessly switch between different drawing tools.
*   **Responsive UI:** A clean, user-friendly interface designed with XAML.

## Technologies & Architecture

*   **Language:** C#
*   **Framework:** WPF (Windows Presentation Foundation) & XAML
*   **Architecture:** Object-Oriented Programming (OOP)
*   **Design Patterns:**
    *   **Strategy Pattern:** Implemented for the dynamic management and interchangeable behavior of different drawing tools (e.g., swapping between brush and shape tools at runtime).
    *   **Command Pattern:** Encapsulates drawing actions as objects, establishing a solid foundation for robust Undo/Redo functionalities.
